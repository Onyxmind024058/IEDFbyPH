# **RFEA Analyzer – User Manual**

## 1. Introduction

**RFEA Analyzer** is a desktop application designed for the analysis of **Ion Energy Distribution Functions (IEDF)** obtained from **Retarding Field Energy Analyzer (RFEA)** measurements in plasma experiments.

The software allows users to:

- Load RFEA CSV data files
- Compute averaged I–V curves
- Apply configurable smoothing algorithms
- Extract the IEDF
- Estimate plasma parameters (ion flux, density, average energy)
- Compute the **ion transit time to RF period ratio (τᵢ / τ_rf)**
- Visualize and export results

The application is intended for **researchers, engineers, and students** working with low-temperature plasmas.

## 2. System Requirements

- **Operating system:** Windows 10 / 11
- **Standalone executable:** No Python installation required
- **Input format:** CSV files generated by RFEA acquisition systems

## 3. Application Overview

The application window is divided into three main areas:

1. **Control Panel (left)**
    - File selection

2. **Plot Area (right)**
    - I–V curves

3. **Results Panel**

## 4. Loading Data

### 4.1 Opening a CSV file

1. Click **“Open CSV…”**
2. Select an RFEA CSV file using the file explorer
3. The file path will appear in the interface
4. The analysis runs automatically

### 4.2 CSV File Structure

The CSV file must contain:

- Electrode voltage (header line)
- Ion flux reference value
- Multiple I–V traces separated by blank or NaN rows

(Compatible with the MATLAB RFEA acquisition format.)

---

## 5. Analysis Parameters

### 5.1 Pressure

- **Units:** same as acquisition (typically Pa or mTorr)
- Used for ion flux correction

### 5.2 Smoothing of dI/dE

- **SmoothFactordIdV**
- Controls smoothing applied to the derivative used for the IEDF
- Higher values → smoother IEDF, less noise

### 5.3 I–V Smoothing Method

Choose one of the following:

### Recursive (recommended)

- Optimized for single-peak IEDF
- Automatically detects the main energy peak
- Best default choice

### Moving Average (Mavg)

- Simple moving average
- Adjustable window size

### Savitzky–Golay (SG)

- Polynomial smoothing
- Parameters:
    - Polynomial order
    - Window length (odd number)

## 6. Plasma & Sheath Parameters (τ Ratio)

These parameters are used to compute the **ion transit time to RF period ratio**:

### 6.1 Vp – Plasma Potential

- **Units:** Volts (V)
- Typical range: 10–50 V

### 6.2 Mi – Ion Mass

- **Units:** atomic mass unit (u)
- Examples:
    - Argon: 40
    - Helium: 4
    - Oxygen: 16

### 6.3 f_rf – RF Frequency

- **Units:** Hz
- Default: 13.56 MHz

### 6.4 Te – Electron Temperature

- **Units:** eV
- Default: 3 eV

### 6.5 α – Sheath Expansion Factor

- Dimensionless
- Default: 3

---

## 7. Output Quantities

The application computes and displays:

- **Eavg** – Average ion energy (eV)
- **Flux** – Ion flux (m⁻²·s⁻¹)
- **ni** – Ion density (m⁻³)
- **Epeak** – Energy at IEDF maximum (eV)
- **τᵢ / τ_rf** – Ion transit time to RF period ratio
- **Electrode voltage**
- **Imported ion flux reference**

---

## 8. Plot Description

### 8.1 Left Axis (Current)

- **Black curve:** Averaged I–V curve (`I`)
- **Green curve:** Smoothed I–V curve (`Ismooth`)

### 8.2 Right Axis (IEDF)

- **Red curve:** Ion Energy Distribution Function (dI/dE)

### 8.3 Interactivity

The plot supports:

- Zoom
- Pan
- Reset view
- Save image
- Cursor hover readout (energy and values)

---

## 9. Exporting Results

### 9.1 Export Data (CSV)

Click **“Export data (CSV)…”** to save:

- Vavg
- Iavg
- Ismooth
- E
- IEDF (dI/dE)

All arrays are aligned and padded if needed.

### 9.2 Export Plot

Click **“Export plot (PNG/SVG)…”** to save:

- High-resolution PNG (publication ready)
- SVG vector format (recommended for figures)

## 11. Typical Workflow

1. Open CSV file
2. Verify I–V and IEDF shapes
3. Adjust smoothing parameters
4. Set plasma/sheath parameters
5. Check τᵢ / τ_rf value
6. Export data and figures

---

## 12. Notes & Recommendations

- Start with **Recursive smoothing**
- Avoid excessive smoothing of dI/dE
- Ensure Vp and Mi values match experimental conditions
- Use SVG export for publications
